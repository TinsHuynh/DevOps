# Các chức năng cần chỉnh sửa (Đã hoàn thành toàn bộ ✓)

## Tài khoản & quyền 
- [x] Chỉnh sửa hiển thị tài khoản 
- [x] Cho phép admin thêm sửa xoá tài khoản và phân quyền 
- [x] Quản lý danh mục thêm ngành học để chọn được ngành học khi thêm sinh viên  
- [x] Quản lý danh mục thêm lớp học để chọn được lớp học khi thêm sinh viên
- [x] Quản lý danh mục thêm khoa để chọn được khoa khi thêm giáo viên hoặc học sinh
